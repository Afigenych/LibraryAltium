# Библиотека компонентов для Altium Designer
## Правила оформления библиотеки
* Единицы измерения - мм.
* Использовать шрифт ГОСТ тип А из файла "ГОСТ тип А.ttf", который можно найти в репозитории.
* Размер кегля - 10 для нумерации пинов, 14 для всего остального.
* Сетка УГО компонентов - 2,5 мм.

### Как пользоваться библиотекой

1. В случае, если у вас нет гит-клиента:

1.1.  Скачать архив с файлами библиотеки, нажав кнопку "Clone or download", расположенную справа над списком файлов репозитория, и выбрав пункт "Download ZIP".
1.2.  Разархивировать скачанные файлы в папку, путь к которой не содержит национальных символов и пробелов.
1.3.  Запустить Altium Designer.
1.4.  Открыть панель Libraries, нажать кнопку "Libraries...", выбрать вкладку "Installed".
1.5.  Нажать на кнопку "Install...", из выпадающего списка выбрать "Install from file...".
1.6.  В окне поиска из выпадающего списка выбрать расширение файла "*.DBLIB".
1.7.  В корневой папке библиотеки найти и выбрать файл "GOST.DbLib".
1.8.  Нажать "Открыть" или "Open".
1.9.  Нажать "Close".
1.10. Пользоваться : )

2. В случае, если у вас есть гит-клиент:

2.1. Нажать кнопку "Clone or download", скопировать появившуюся ссылку на репозиторий для гит-клиента.
2.2. Клонировать репозиторий через гит-клиент на локальную машину.
2.3. Установить библиотеку, повторив пункты 1.3 -- 1.10.
