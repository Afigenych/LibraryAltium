# Библиотека компонентов для Altium Designer
# ООО "Контур-НИИРС"

## Правила оформления библиотеки
	1. Единицы измерения - мм.
	2. Использовать шрифт ГОСТ тип А из файла "ГОСТ тип А.ttf", который можно найти в репозитории.
	3. Размер кегля - 10 для нумерации пинов, 14 для всего остального.
	4. Сетка УГО компонентов - 2,5 мм.
